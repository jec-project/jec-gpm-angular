About Lazy Loading

Lazy loading in means register the module in the angular application only when the route hits!

In this bootstrap application, lazy loading sample parts of are isolated in each lazyLoading folder of the webbapp/app structure. The current view, associated with RootComponent, is loaded only when the user navigates to the lazy-loading/ route.
